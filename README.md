# zhportal-leyu-hub

这是一个用于归档和发布多个独立 HTML 页面的仓库。仓库中存放的页面内容不针对任何特定域名或具体网站，仅作为通用页面存档与分发用途。

## 项目简介

zhportal-leyu-hub 旨在提供一个集中存放静态 HTML 页面的空间。这些页面可能包含信息展示、工具页面、活动页面等，彼此独立且不依赖特定后端服务。仓库本身不包含动态功能，所有页面均为纯前端 HTML 文件。

## 目录结构说明

```
zhportal-leyu-hub/
├── pages/          # 存放独立 HTML 页面文件
├── assets/         # 页面引用的静态资源（如 CSS、JS、图片等）
├── docs/           # 文档或说明文件（可选）
└── README.md       # 本文件
```

- **pages/**：每个 HTML 文件为一个独立页面，文件名通常反映页面用途或内容主题。
- **assets/**：按页面或功能组织，包含样式、脚本、字体、图标等资源。
- **docs/**：存放与仓库或页面使用相关的说明文档。

## 页面归档说明

仓库中的页面可能以不同版本或时间顺序归档。每个页面在其文件头部或附带的说明文件中会标记创建日期、版本号及简要描述。页面之间无依赖关系，可单独访问。

归档页面仅供查阅或引用，不保证长期更新。使用者可根据需要自由下载或部署。

## 使用方式

1. 克隆或下载本仓库：
   ```bash
   git clone https://github.com/your-username/zhportal-leyu-hub.git
   ```
2. 直接在浏览器中打开 `pages/` 目录下的任意 HTML 文件即可查看页面内容。
3. 如需部署到服务器，将整个仓库或 `pages/` 目录上传至静态托管服务即可。
